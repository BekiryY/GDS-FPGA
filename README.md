# GDS-FPGA

The purpose of this repository is to create a custom FPGA using hand-drawn standard cells, and to provide the capability to program it within a simulation environment.

## Overview
- **Custom Hardware**: Building an FPGA architecture using custom, hand-drawn standard cells.
- **Simulation Flow**: Includes the necessary environment and testbenches to program the custom FPGA fabric, allowing logic implementation and verification entirely in simulation.
